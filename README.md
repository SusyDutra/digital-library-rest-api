# Digital Library REST API

## Docker Execution

### Docker Compose

```bash
# Put the services up
docker-compose up -d

# See logs
docker-compose logs -f

# Put the services down
docker-compose down
```

### Endpoints

- API: http://localhost:8000
- Docs: http://localhost:8000/docs
- PostgreSQL: localhost:5432
